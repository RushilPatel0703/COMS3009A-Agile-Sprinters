Final User Stories:
1. As a user, I want to be able input another user's phone number into a box, which will send them a friend request, so that I will be able to talk to people I know. 
2. As a user, I want to be able to view all requests that other users have sent me so that I can choose to either accept or decline them and chat to people I know. 
3. As a user, I want to be able to press a button that will allow me to compose a new message so that I can start a conversation with my current contacts as well as potentially new contacts effortlessly. 
4. As a user, I want to be able to upload either an image or an image with a caption to my story for 24 hours with a display time of 5 seconds so that my contacts can view my story and see what I am posting about.
5. As a user, I want the image from my status to be displayed as my profile picture for 24 hours so that my contacts will know when I have posted something to my story.  
6. As a user, I want to be able to see the number of the  image I am currently previewing as a fraction of the total images selected when sending multiple image so that I can see exactly how many images I have selected to send to another user.
